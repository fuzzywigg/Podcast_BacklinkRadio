### File Organization

```
/episodes
  /ep01-athlete-name
    - notes.md (show notes)
    - transcript.txt
    - assets/ (clips, images)
  episode_template.md

/outreach
  - guest_tracker.csv (master CRM)
  - pitch_templates.md

/assets
  - logo/
  - thumbnails/
  - music/

/scripts
  - intro_outro.md
  - ad_reads.md
```

### Episode Pipeline

1. **BOOKED** → Guest confirmed, date set
2. **RESEARCHED** → Background prep complete
3. **RECORDED** → Raw audio captured
4. **EDITED** → Post-production done
5. **APPROVED** → Guest sign-off (if needed)
6. **SCHEDULED** → Uploaded, release date set
7. **LIVE** → Published + promoted

### Commit Messages
Use prefixes:
- `[EP]` - Episode content
- `[OUTREACH]` - Guest booking
- `[ASSET]` - Branding/design
- `[DOCS]` - Documentation

Example: `[EP] Add show notes for Chloe Kim interview`

### Tools
- **Recording:** Riverside.fm / Zencastr
- **Editing:** Descript / Audacity
- **Hosting:** Buzzsprout / Spotify for Podcasters
- **Social:** ViewCreator.ai / Opus Clip
- **Tracking:** This repo + guest_tracker.csv
