# Alignment & Andon Labs Integration

This directory contains the scaffolding and implementation for "Alignment Tasks" using the [Andon Labs Claude Code SDK](https://github.com/AndonLabs/claude-code-sdk-python-andon-special).

## Purpose
To integrate advanced AI capabilities, specifically:
*   **In-Process MCP Servers:** For custom tools that run within the Hive's process.
*   **Hooks:** For deterministic processing and automated feedback loops to reinforce the `ConstitutionalGateway`.

## Setup
Install the dependencies:
```bash
pip install -r requirements.txt
```
